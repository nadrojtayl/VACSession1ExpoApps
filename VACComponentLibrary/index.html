Panels


Panels is a component that allows you to create an area with content determined when the user clicks on a top row of buttons

Prop Name Type Required Description Example
options Array Yes An array that defines how many different buttons will appear on top of the panel. Each one will display different content. ["MenuA","MenuB","MenuC"]
Panel1 (Or Panel2...Panel3...Panel4...) JSX Yes JSX for the content that should appear in each panel panel0 = { ( PANEL 1 CONTENT ) }
ButtonStyle StyleObject No Sets the style for the menu buttons at the top of the panel panel0 = { ( PANEL 1 CONTENT ) }

DropdownContent


DropdownContent is a component that allows you to create an area with content determined when the user selects a dropdown

Prop Name Type Required Description Example
options Array Yes An array that defines how many different options will appear in the top dropdown. Each one will display different content. ["MenuA","MenuB","MenuC"]
Panel1 (Or Panel2...Panel3...Panel4...) JSX Yes JSX for the content that should appear in each panel panel0 = { ( PANEL 1 CONTENT ) }
DropdownStyle StyleObject No Sets the style for the dropdown {width:"100%"}

CollectInfo


CollectInfo is a component that creates a button which raises a popup to send information to a remote server and database. The schema for the server is determined programatically

Prop Name Type Required Description Example
url String Yes A url to your heroku database setup from VAC "http://www.my_server.com/table"
ContainerStyle StyleObject Yes Styling for the popup container that gets information {width:"100%",alignItems:"center"}
ButtonStyle StyleObject No Sets the style for the dropdown at {width:"100%",alignItems:"center"} }
